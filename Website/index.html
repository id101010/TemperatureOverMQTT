EMBSY Embedded Systems Miniproject

Projects

This project aims to collect and visualize temperature and acceleration data using a Bluetooth-Low-Energy sensor connected to a RaspberryPi. To visualize the aquired data we use google charts.

Please Connect to the MQTT Broker to get data from the sensor

Press to send testdata to the MQTT Broker and get them automatically back to the Chart, "connect" necessary

Press to stop the visualisation mode to analyze the data

Press to start the visualisation again. Start mode ist the default mode.
